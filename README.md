# Badminton_Analysis

This repository is used for hosting and deploying the Badminton Posture Analysis System.

## Deployment Details
- Hosting Platform: Streamlit Community Cloud
- Main Entry Point: main.py
- System Dependencies: Defined in packages.txt
- Python Dependencies: Defined in requirements.txt

## Project Overview
This project utilizes MediaPipe and OpenCV for badminton pose detection and movement analysis.
